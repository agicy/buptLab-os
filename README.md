# buptLab-os

这个仓库包含了北京邮电大学 2024-2025 秋季学期《操作系统（叶文）》课程实验的相关代码和报告。

## 使用说明

课程实验共六个，分别为：

1. 进程创建与管理；
2. 线程创建、管理与通信；
3. 进程间通信；
4. 进程/线程的同步互斥；
5. 多核多线程编程及性能分析；
6. 观察进程地址空间。

实验主要在 Ubuntu 24.04 下进行，所需的配置步骤见实验报告。

.
